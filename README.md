# DevOps Hexagon Layout

A modern, visually stunning web interface featuring a dynamic hexagon grid and a custom-animated infinity loop logo. Built with pure HTML5, CSS3, and SVG for maximum performance and aesthetics.

- **Dynamic Hexagon Grid**: 8 interactively styled hexagons housing DevOps-related icons (Azure, GitLab, Sync, Helm, Rocket, etc.).
- **Interlocking Infinity Logo**: A custom-engineered SVG infinity loop with:
  - **Circular Design**: Perfectly balanced "round" (gul) loops.
  - **Interlocking Gaps**: Animated mask-based arrow gaps (`>`) that move along the path.
  - **Smooth Gradients**: 8 distinct segment gradients creating a vibrant, fluid motion.
- **Glassmorphic Aesthetics**: Modern dark mode design with subtle glows, gradients, and hover micro-animations.
- **Responsive Design**: Fully responsive layout that adapts to different screen sizes.
